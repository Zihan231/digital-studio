## 🌟 Overview

**Digital Studio** is a high-end, responsive portfolio website built specifically for a modern digital printing business. It features a stunning UI designed with premium layout spacing, elegant typography, and a cohesive aesthetic to leave a lasting impression on potential clients. The project embraces internationalization (i18n) to support both English and Bengali audiences flawlessly.

## ✨ Features

- 🎨 **Premium Aesthetic**: "Thousand-dollar" design with standardized vertical padding, seamless section transitions, and cohesive color palettes.
- 📱 **Mobile Responsive**: Flawless experience across all devices with globally center-aligned headers and optimized typography for smaller screens.
- 🌐 **Bilingual Support (i18n)**: Fully localized in **English** and **Bengali (Hind Siliguri)**, maintaining professional line-heights and text formatting.
- ⚡ **High Performance**: Built on Next.js App Router for optimal speed, SEO, and user experience.
- 🖼️ **Dynamic Project Galleries**: Engaging project showcases, team portfolios, and machinery overviews.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Library**: [React](https://react.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Localization**: `i18next` & `react-i18next`

## 🚀 Getting Started

Follow these steps to set up the project locally:

### Prerequisites
- **Node.js**: `v18.x` or higher
- **npm** / **yarn** / **pnpm**

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Zihan231/digital-studio.git
   cd digital-studio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open the App:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your preferred browser.
